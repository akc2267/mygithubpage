Clutch

Clutch is an iOS app I'm currently working on for a Mobile App Design class. It's available in the app store. There are four other students working on this project. Two computer science majors and two journlism majors. For this project I focused on user experience, ui design, and coding. The goal of this semester long project was to work on an interdisciplinary team that focused on marketing (social media), branding, and creating a succesful product.

Brainstorming and Research:

We were given a week to come up with an app proposal for this class. We brainstormed many ideas but found that one of the pain points everyone on our team shared, was that we felt the available resources for tracking the buses in austin were not great. We researched available bus apps in austin and other cities to see how others had solved this problem. We also research the available resources from Capital Metro for getting real-time bus locations.

UI Sketches and early implementation:

We started by deciding on a feature set for our MVP. Once this set had been decided, we set goal for our first sprint, then began to work. We searched for a data source for route times. We began to work on how our data would be structued and how the app would display that data to users.
